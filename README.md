# Financial Invoice Analysis SQL

SQL project focused on invoice, payment and customer analysis.

## Project overview
This project contains 12 SQL queries designed to analyze a billing database.  
The analysis includes:
- number of customers, invoices, payments and payment methods
- total revenue and total cash collected
- outstanding balances on unpaid invoices
- customer-level revenue and residual credit
- payment delays
- payment method analysis
- monthly revenue trends
- collection rate percentage

## File included
- `financial_invoice_analysis.sql`: main SQL script with 12 analysis queries

## Tools used
- MySQL
- SQL
- GitHub

## Main objectives
- Monitor invoice performance
- Evaluate residual credit exposure
- Analyze collection efficiency
- Support financial and accounting analysis
